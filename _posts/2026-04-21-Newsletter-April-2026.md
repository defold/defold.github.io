## Defold 1.12.3 Has Been Released

![Defold 1.12.3 Has Been Released](/images/posts/newsletter-april-2026/1123.png)

Highlights:

- Time exposed to shaders
- Texture Transform semantic, useful for scrolling textures
- Script Structure: list of functions and variables in the sidebar
- Duplicate text selection
- Filters in Input Bindings to assign inputs to actions more easily
- New Properties API for Editor Scripts
- OpenAPI for the Editor HTTP Server

And many more.

[Check Release Notes](https://forum.defold.com/t/defold-1-12-3-has-been-released/82619)
