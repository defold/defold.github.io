How to go Pro?

Serious game development deserves serious-sounding features. Defold was engineered to support a Pro version from ground up - install the custom Editor Creator Studio Suite (Editor CSS), and the editor will load it on startup from your local .defold folder. The royal editor treatment can leave this page and enter your workstation now.

  1. Download the Editor Creator Studio Suite (Editor CSS).
  2. Create your local Defold configuration folder. Make a folder named .defold in your home directory.
    • Windows: C:\Users\Your Username\.defold
    • macOS: /Users/Your Username/.defold
    • Linux: ~/.defold
  3. Place the downloaded Editor Creator Studio Suite (Editor CSS) file in that folder. Make sure it is named editor.css and is inside .defold. If you already have one, back it up first unless you enjoy premium merge conflicts.
  4. Start or restart the editor. Defold will load the Editor Creator Studio Suite and upgrade Defold to Pro on startup.
  5. Enjoy the consequences. The future is now.
