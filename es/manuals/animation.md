# Animación

Defold tiene soporte integrado para muchos tipos de animación que puedes usar como fuente de gráficos para componentes:

* [Animación flip-book](/es/manuals/flipbook-animation) - Reproduce una serie de imágenes fijas en sucesión
* [Animación de modelo](/es/manuals/model-animation) - Reproduce animaciones 3D con skin
* [Animación de propiedades](/es/manuals/property-animation) - Anima propiedades como posición, escala, rotación y muchas otras

Se pueden agregar formatos de animación adicionales mediante extensiones:

* [Animación Rive](/extension-rive) - Reproduce animaciones esqueléticas 2D basadas en vectores
* [Animación Spine](/extension-spine) - Reproduce animaciones esqueléticas 2D con texturas
