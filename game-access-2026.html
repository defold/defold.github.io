Defold at a glance

Defold is a free, source-available game engine for building 2D and 3D games from one compact production workflow. It includes a visual editor, Lua scripting, runtime, scene and GUI tools, particle and tilemap editors, debugger, profiler, build pipeline, asset system, native extensions, editor scripting, and deployment to web, mobile, desktop, and supported console platforms.

Built for shipping games

Teams choose Defold when download size, performance, fast iteration, and clear commercial terms matter. The engine is free to use, source available, extensible, and actively released on a predictable monthly cycle.

  • 2D and 3D scenes, GUI, particles, shaders, and custom rendering
  • Make games and ship them to all supported platforms
  • No royalties, no fees - completely free
  • Reach players - 600M+ plays for Defold games on Poki in 2024
  • Keep control - source available under a developer-friendly license
  • Write game logic using Lua, enhance it with native extensions
  • Customize both engine and editor however you like

Why teams choose Defold

Small engine. Fast workflow. Clear terms. Defold is built around the daily decisions that affect production speed, build size, and release planning.

Start immediately, no setup

No setup marathon, no registration gate, no separate SDK stack before you can try an idea.

  • Editor, runtime, scripting, debugger, and build tools included
  • Lua workflow with native extensions when you need deeper control
  • Fast builds and hot reload for quick iteration
  • Free forever, source available, no royalties

Reach many platforms with one project

Keep one codebase for web, mobile, desktop, and supported console targets.

  • HTML5, iOS, Android, Windows, macOS, Linux, Steam, Poki, and more
  • Small builds help web and mobile games load quickly
  • Live Update support for content delivery after release

Keep projects flexible and lightweight

Defold stays small by default and gives teams extension points when a project needs deeper control over engine features, SDKs, tools, or asset workflows.

  • Small size - only 1 MB empty HTML5 engine size, gzipped
  • Cut out engine parts with a single App Manifest file
  • Native extensions for SDKs, services, and platform-specific code
  • Editor scripts and custom asset pipelines
  • Monthly releases and a practical open ecosystem with source available
