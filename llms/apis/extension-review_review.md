# extension-review

**Namespace:** `review`
**Language:** Lua
**Type:** Extension

Functions and constants for interacting with review APIs

## API

### review.request_review
*Type:* FUNCTION
Open native review/rating popup

### review.is_supported
*Type:* FUNCTION
Available only on iOS 10.3+. Android 5.0+ (API 21+) and the Google Play Store has to be installed.
