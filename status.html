What this page checks

This page reads the public health_report endpoints exposed by the production and stage build servers, then summarizes the platform builders returned by each endpoint.

A stage issue does not necessarily affect production, but it can signal incoming maintenance, rollout work, or a partial environment problem.

Need help?

If the server looks healthy but your remote build still fails, check the build log first and then use the support channels or the source code discussion area for troubleshooting.
